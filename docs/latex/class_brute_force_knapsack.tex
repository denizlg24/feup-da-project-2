\doxysubsection{Detailed Description}
Implements the brute-\/force approach to solve the 0/1 Knapsack problem. 

This class derives from the \doxylink{class_command}{Command} interface and overrides the execute method to evaluate all possible combinations of items to find the optimal selection that maximizes value without exceeding the knapsack\textquotesingle{}s capacity.
