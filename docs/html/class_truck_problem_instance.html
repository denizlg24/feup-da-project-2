Detailed Description

Encapsulates a truck loading problem instance with pallets, capacity, and related data.

This class manages the data related to a truck loading optimization problem, including the truck's capacity, the number of pallets, and the pallets themselves. It provides methods to add pallets, set/get the capacity and number of pallets, and retrieve vectors of weights and profits for use in optimization algorithms.
